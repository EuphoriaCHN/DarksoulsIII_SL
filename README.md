## 黑暗之魂 III SL 脚本

> 仅适用于 Windows 10

### Use

1. 在游戏开始界面，双击运行 `save.bat`
2. 进入游戏，快乐 PY
3. 保存退出游戏，回到游戏开始界面（**点击任意键继续**），双击运行 `load.bat` 即可
